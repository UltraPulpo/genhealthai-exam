## Background

The goal of this assessment is to evaluate your ability to design, implement, and deploy a live REST API. The focus is on delivering a Minimum Viable Product (MVP) that is practical, functional, and ready to get off the ground quickly in a time-constrained environment.

Your goal should be to make this the most amazing, feature complete, and scalable solution. All code should work. Bonus points for going beyond these features.

## Task

Complete a REST API (and frontend application if time permitting) with the following:
● Setup endpoints to perform CRUD operations for an “Order” entity
● Persist values to a database (This may be a local database for simplicity although remote databases are fine too)
● Accept an uploaded document via POST request and extract the patient’s First name, last name and date of birth (we will test this with a PDF you have never seen)
● Log all user activity to a database
● Deploy this solution somewhere that is publicly accessible
